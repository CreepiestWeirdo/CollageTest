# CollageTest

Con assets de collages brindados por el artista Poli Romero, realizaré un prototipo de juego de un equilibrista yendo de un lado a otro esquivando zarpazos de un gato negro. 

Cada vez que el equilibrista llegue a un extremo, el gato se pondrá más agresivo atacando más frecuentemente. 

Cada vez que se llega a un extremo, se suma puntos.
